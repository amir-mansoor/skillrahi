# Code of Conduct

## Our Pledge

We as contributors and maintainers pledge to make participation in this project and our community a harassment-free experience for everyone.

We commit to a welcoming, safe, friendly space â€” regardless of age, body size, disability, ethnicity, sex characteristics, gender identity and expression, experience level, nationality, personal appearance, race, religion, or sexual identity and orientation.

We want SkillRahi to be a place where beginner developers feel confident to learn and contribute.

## Our Standards

Examples of positive behavior:

- Using respectful and inclusive language
- Being patient with beginners
- Giving constructive feedback instead of personal attacks
- Accepting that people have different opinions and experiences
- Focusing on what is best for the community

Examples of behavior that is not acceptable:

- Harassment, insults, trolling
- Sharing private information without permission (real names, emails, etc)
- Any political, religious, or personal hate speech
- Sexualized language or imagery
- Public or private harassment of contributors

## Responsibilities of Maintainers

Project maintainers have the right and responsibility to:

- Remove comments, commits, code, issues, or PRs that violate this Code of Conduct
- Ban temporarily or permanently any contributor for abusive behavior

## Scope

This Code of Conduct applies:

- Within project issues
- Pull requests
- Discussions
- GitHub community spaces
- Any place where the project is represented

## Enforcement

Violations will be reviewed by maintainers.  
If you see something violating the Code of Conduct, please report by opening an issue with the title:

`[Code of Conduct] Incident`
